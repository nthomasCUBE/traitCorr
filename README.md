## Installation

Following packages are needed, the Rscript "ReporteRs" has been replaced by R package "officer",
as suggestd by [davidgohel/ReporteRs](https://github.com/davidgohel/ReporteRs).
```
library(d3heatmap)
library(gplots)
library(officer)
library(VennDiagram)
library(scales)
library(shiny)
library(shinyalert)
library(shinyBS)
library(shinyjs)
library(shinythemes)
library(xlsx)
```
